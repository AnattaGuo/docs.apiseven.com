# 应用管理

## 名词解释

### 组织

查看开发者管理文档。

### 应用名称

单个应用的名称，帮助开发者识别并了解应用。

### 描述

单个应用的描述，帮助开发者快速了解应用的作用及注意事项。

### 更新时间

更新应用描述之后的时间，如果没有更新过应用描述，则跟应用的创建时间相同。
