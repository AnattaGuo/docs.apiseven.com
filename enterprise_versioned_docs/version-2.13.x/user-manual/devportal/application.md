## 新增应用

应用是由开发者维护的调用API的主体，不支持在管理端创建。

## 删除应用

应用是由开发者维护的调用API的主体，不支持在管理端删除。

## 编辑应用

应用是由开发者维护的调用API的主体，不支持在管理端编辑。
